# VulnScan Lite

Many website owners, especially those with small blogs or
business sites, have no easy way to perform a basic security
health check. VulnScan Lite provides a simple, on demand
scanning service that checks for common, low level web
vulnerabiliti
vulnerabilities without being intrusive.es without being intrusive.

A user enters their website URL, and the service runs a series
of automated checks. It does not perform aggressive
penetration testing but instead focuses on "passive" checks
like insecure HTTP headers, exposed server informatio n, and
basic checks for outdated software versions.

Key Features:

1. Simple Scan Interface: A single input field for the user to
enter their URL and start a scan.
2. Vulnerability Checks:
HTTP Header Analysis: Checks for missing security headers
(Content Sec urity Policy, X Frame Options, etc.).
3. Cookie Attributes: Checks if cookies are set with Secure and
HttpOnly flags.
4. Basic CMS/Framework Detection: Identifies common
platforms (like WordPress, Joomla) and checks against a
database of known vulnerabilities for the detected version.
5. SSL/TLS Configuration: Checks for valid SSL certificates and
strong cipher suites.
6. Easy
to Understand Reports: Presents the results in a clear
report card format, explaining each vulnerability and
providing links to remediation guides.
